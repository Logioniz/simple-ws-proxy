# Прокси сервер, работающий по не шифрованному WS.

## Authentication

Запрос клиента содержит хедеры:
- Time: <timestamp>
- Authentication: SimpleProxy <token>

Проверка времени аутентификации:
- Time должен быть в диапазоне от текущего времени минус 5 секунд до текущего времени плюс 5 секунд. Диапазон задаётся в параметрах запуска сервера.

Токен аутентификации формируется так:
- <token> = hmac(sha256, <time>, <session_secret_key>)
- <session_secret_key> = func(<secret_key>, <time>), func - любая криптографическая функция
- <secret_key> задаются в параметрах запуска

## Шифрование и передача данных

Данные шифруются xor-ом с <session_secret_key>.

## Протокол

Первый запрос (connect) после подключения указывет, к какому сайту коннектиться.

{
    "connect": "<host>:<port>"
}

Далее запросы передаются как есть от клиента к серверу, шифруя xor-ом с <session_secret_key>.

## Клиет

Клиент принимает соединения по socks5 возможно с аутентификацией по user/password и проксирует данные через WS соединение.
